# Market Segmentation & Market Basket Analysis

This Streamlit application performs Customer Segmentation using K-Means clustering (based on RFM analysis) and Market Basket Analysis using Association Rules on the Online Retail II dataset.

## Setup & Run Locally

1.  **Clone or download the repository.**
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the application:**
    ```bash
    streamlit run market_streamlit_app.py
    ```

## Deployment to Streamlit Cloud

1.  Push this repository to GitHub.
2.  Log in to Streamlit Cloud.
3.  Create a new app and select this repository.
4.  Streamlit Cloud should automatically install the dependencies from `requirements.txt`.
